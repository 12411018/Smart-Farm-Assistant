## 📱 Responsive Design

✅ **Mobile First Approach**

| Device | Width | Layout |
|--------|-------|--------|
| Mobile | <480px | 1 column |
| Phone | 480-768px | 1-2 columns |
| Tablet | 768-1200px | 2-3 columns |
| Desktop | 1200px+ | Full grid |

All pages tested and optimized for each breakpoint.

## 🛠️ Technology Stack

```
Frontend: React 18.x (Functional Components)
Routing: React Router v6
Styling: Plain CSS with Variables
Build: Vite
Server: Node.js + Vite Dev Server
Package Manager: npm
```

### No External UI Frameworks ❌
- ❌ Tailwind CSS
- ❌ Bootstrap
- ❌ Material UI
- ❌ Styled Components
✅ **Plain CSS Only**
